## Claim Boundaries

- Agent BlackBox Arena is an OpenEnv-style repair environment for agent reliability CI.
- The environment is not an observability dashboard; it trains replay, evidence selection, repair patching, regression testing, and certificate gating.
- Base 0.5B failed strict JSON on standard eval.
- Challenge-curriculum SFT fixed JSON validity and recovered nonzero evidence/certificate success on shuffled/blinded challenge prompts.
- Qwen3-4B final H200 SFT+GRPO is the selected final model and beat the 0.5B fallback on reported challenge evidence/certificate metrics.
- Hidden regression and valid-preservation metrics are verifier-scored and deterministic.
- The certificate is bounded to this symbolic benchmark, not a global safety proof.
